# 创建组和用户<a name="ZH-CN_TOPIC_0231470849"></a>

>![](public_sys-resources/icon-note.gif) **说明：**   
>服务器环境下，为了系统安全，通常会为进程分配单独的用户，以实现权限隔离。本章节创建的组和用户都是操作系统层面的，不是数据库层面的。  

1.  创建MySQL用户（组）。

    ```
    #groupadd mysql
    ```

    ```
    #useradd -g mysql mysql
    ```

2.  设置MySQL用户密码。

    ```
    #passwd mysql
    ```

    重复输入密码（根据实际需求设置密码）。
