# 安装前准备<a name="ZH-CN_TOPIC_0186524175"></a>

## 2.1 环境要求<a name="zh-cn_topic_0176377961_section1128221214910"></a>

HCC支持如下64位操作系统，预留1GB以上硬盘空间。

-   SUSE12 SP4及以上版本。
-   Ubuntu 16.04及以上版本。
-   其他glibc版本在2.22及以上的Linux发行版。
