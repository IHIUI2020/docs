## OS Requirements<a name="section1092214563422"></a>

The openEuler OS is required.

For details about how to install the openEuler OS, see the  _openEuler 20.03 LTS Installation Guide_. On the  **SOFTWARE SELECTION**  page, select  **Development Tools**  in the  **Add-Ons for Selected Environment**  area.
